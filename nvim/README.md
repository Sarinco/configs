## shortcuts

C-s : save all files
<leader>r : replace in file (g = global, c = selector, i = case insensitive)
<leader>yy : yank in clipboard
<leader>pp : paste from clipboard

### telescope
C-p : find file
<leader>fg : live grep in directory
